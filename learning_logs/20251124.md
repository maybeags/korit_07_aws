# 입실 체크 해주세요 !! 💌

1. AWS 프리티어 계정 생성하기

-   https://aws.amazon.com/ 접속
-   우측 상단 `계정 생성` 버튼 클릭
-   sign up으로 들어가서,
-   Root user email address에 email 입력
-   AWS account name에 계정 이름 등록
-   Verify email address 버튼 클릭
-   등록한 email로 가서 verify 과정 수행
-   root password 등록
-   연락처 정보 입력(주소를 영어로 작성- 한국어가능한가봅니다)
-   결제 정보 입력
-   자격 증명 창이 뜨면 폰 번호 입력
-   free tier 혹은 기본 지원 - 무료 플랜 선택
